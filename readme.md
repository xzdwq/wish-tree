### Installation

- `git clone <url> -b <branch-name>`
- `pnpm i`

---

### Development mode
- `pnpm dev`
  - app: `http://localhost:4444`
  - api: `http://localhost:7000`

### Docker
- `docker-compose -f docker/development/docker-compose.yml up --build`
  - app: `http://localhost:4445`
  - api: `http://localhost:7000`
